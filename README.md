## Flexbox

A way to arrange elements easier with CSS.
* Assigned to parent element (directs child elements)
* Can only be a row or a column

*Example:*
```
<div id="parent">
    <div></div>
    <div></div>
    <div></div>
</div>

#parent {
    display: flex;
    flex-direction: row;
}
```
